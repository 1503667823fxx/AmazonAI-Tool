# A+ 模板智能搜索使用指南

## 🔍 搜索功能说明

### 核心特性
智能搜索引擎支持多种搜索方式，帮助你快速找到最适合的模板。

## 📝 搜索示例

- 组合搜索: `南瓜 派对 服装`

**搜索结果**:
- ⭐ 万圣节恐怖风 (节日匹配: 万圣节)
- 相关关键词: 南瓜、鬼魂、巫师、万圣节服装、面具

**为什么匹配**:
- 模板包含"南瓜"关键词
- 节日属性为"万圣节"
- 标签包含"服装"、"派对"
- 风格适合恐怖/神秘主题

## 🎯 搜索技巧

### 关键词选择
1. **产品核心词**: 直接描述产品的词 (如: 耳机、面膜、玩具)
2. **场景词**: 使用场景 (如: 派对、户外、家居)
3. **节日词**: 相关节日 (如: 万圣节、圣诞节、春节)
4. **风格词**: 设计风格 (如: 复古、现代、可爱)
5. **功能词**: 产品特点 (如: 防水、智能、手工)

### 搜索策略
- **从宽到窄**: 先用大类别搜索，再用具体关键词
- **组合搜索**: 使用多个关键词提高精准度
- **查看建议**: 注意搜索建议中的相关词
- **使用筛选**: 配合类别和节日筛选器

### 匹配度说明
- **⭐ 高匹配** (得分 > 10): 多个关键词匹配，强烈推荐
- **中等匹配** (得分 5-10): 部分关键词匹配，可以考虑
- **低匹配** (得分 < 5): 风格相似，可作为备选

## 🤖 AI 智能推荐

### 使用方法
1. 填写产品名称、类别和卖点
2. 点击"获取AI推荐模板"按钮
3. 查看推荐结果和推荐原因
4. 点击推荐的模板直接应用

### 推荐逻辑
AI 会综合分析:
- 产品名称中的关键词
- 产品类别
- 产品卖点和特色
- 模板的适用场景
- 历史匹配数据

### 推荐示例
**产品信息**:
- 名称: 万圣节南瓜服装
- 类别: 服装配饰
- 卖点: 派对装扮、恐怖造型、节日氛围

**AI 推荐**:
1. 万圣节恐怖风 - 节日主题完美匹配
2. 复古怀旧风 - 适合特殊服装展示
3. 运动活力风 - 适合展示穿着效果

## 🔧 高级功能

### 筛选器组合
- **类别 + 搜索**: 在特定类别中搜索
- **节日 + 搜索**: 在节日模板中搜索
- **类别 + 节日**: 精确定位特定场景

### 相似模板推荐
选择模板后，系统会自动推荐相似模板:
- 相同类别的其他风格
- 相同风格的其他类别
- 相似配色方案的模板

### 搜索建议
输入 2 个字符后，系统会提供:
- 热门关键词
- 相关标签
- 节日名称
- 产品类别

## 📊 搜索评分机制

### 得分权重
- 节日匹配: 15 分 (最高)
- 精确关键词: 10 分
- 标签匹配: 8 分
- 类别匹配: 7 分
- 季节匹配: 6 分
- 风格属性: 5 分
- 文本相似度: 3 分
- 部分词匹配: 2 分

### 如何提高匹配度
1. 使用模板配置中的关键词
2. 包含节日或季节信息
3. 明确产品类别
4. 使用多个相关词组合
5. 参考搜索建议

- **产品类别**: 科技、美妆、家居、运动、母婴
- **设计风格**: 现代、优雅、可爱、活力、复古

- `#优雅` → 优雅风格模板
