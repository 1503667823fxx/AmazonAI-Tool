# 🖌️ Magic Canvas 使用说明

## 功能介绍
Magic Canvas 是一个AI智能重绘工具，让你可以在图片上涂抹指定区域，然后由AI根据你的描述重新绘制该区域。

## 使用步骤

### 1. 上传图片
- 点击「上传原图」按钮
- 支持 PNG、JPG、JPEG、WEBP 格式
- 图片会自动缩放到合适大小（最大600px）

### 2. 涂抹区域
- 在画布上用鼠标涂抹你想要修改的区域
- 红色区域表示将被AI重绘的部分
- 可以调整画笔大小（10-80像素）
- 圆形光标帮助你判断涂抹范围

### 3. 确认涂抹
- 涂抹完成后，点击「✅ 确认涂抹」按钮
- 系统会生成涂抹遮罩数据

### 4. 输入重绘描述
- 在文本框中描述你希望AI在涂抹区域绘制的内容
- 例如："一只可爱的小猫"、"蓝色的天空"、"现代化的建筑"
- 描述越详细，AI生成的效果越准确

### 5. 开始重绘
- 点击「🎨 开始重绘」按钮
- AI会根据你的描述重新绘制涂抹区域
- 等待处理完成

### 6. 查看结果
- 重绘完成后会显示新的图片
- 可以下载保存结果图片
- 如果不满意，可以重新涂抹和描述

## 使用技巧

### 涂抹技巧
- **精确涂抹**：仔细涂抹需要修改的区域边界
- **适中画笔**：选择合适的画笔大小，太大会影响精度
- **完整覆盖**：确保要修改的区域完全被红色覆盖

### 描述技巧
- **具体描述**：使用具体的形容词，如"鲜艳的红色玫瑰"而不是"花"
- **风格指定**：可以指定艺术风格，如"卡通风格的"、"写实的"
- **环境描述**：包含环境信息，如"在阳光下"、"在室内"

### 常见应用场景
- **物体替换**：将图片中的某个物体替换为其他物体
- **背景修改**：改变图片背景
- **细节优化**：修改图片中的小细节
- **创意合成**：在现有图片基础上添加创意元素

## 注意事项

1. **图片质量**：上传高质量的原图能获得更好的重绘效果
2. **涂抹精度**：涂抹区域的精确度直接影响最终效果
3. **描述准确性**：清晰准确的描述是获得理想效果的关键
4. **处理时间**：复杂的重绘可能需要较长处理时间，请耐心等待

## 故障排除

### 涂抹数据消失
如果粘贴涂抹数据后消失，请：
1. 确保数据完整复制
2. 点击确认按钮前不要进行其他操作
3. 如果问题持续，请刷新页面重试

### 重绘效果不理想
如果重绘效果不满意，可以：
1. 重新精确涂抹目标区域
2. 修改描述，使其更加具体
3. 尝试不同的描述风格
4. 调整涂抹区域的大小

### 处理失败
如果重绘处理失败：
1. 检查网络连接
2. 确认图片格式支持
3. 重新上传图片并重试
4. 联系技术支持

## 技术说明

Magic Canvas 使用先进的AI图像修复和生成技术：
- **智能遮罩识别**：精确识别用户涂抹的区域
- **上下文理解**：AI理解图片整体内容和涂抹区域的关系
- **自然融合**：生成的内容与原图自然融合，无明显边界
- **高质量输出**：保持原图质量的同时生成高质量的重绘区域
